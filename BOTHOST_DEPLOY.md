# Быстрый запуск на Bothost

- Рабочая директория кода внутри образа: `/usr/src/app`.
- Постоянная база: `/app/data/database.db`.
- Volume должен монтироваться именно в `/app/data`, не в `/usr/src/app` и не во весь `/app`.
- Команда запуска уже указана в Dockerfile: `python demerzel.py`.
- Приложение слушает `0.0.0.0:$PORT`.

Перед первым запуском новой версии обязательно остановите старую копию бота. Один Telegram-токен не может одновременно использоваться двумя `getUpdates`-процессами.

После запуска откройте `/health`. При проблеме база вернёт список отсутствующих таблиц, а traceback HTTP 500 будет в журнале Bothost.
